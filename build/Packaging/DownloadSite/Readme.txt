Documentation
=============

Agent install documentation can be found at https://docs.newrelic.com/docs/agents/net-agent/installation/new-relic-net-agent-install-introduction.

File Descriptions
=================

Assets for installing the agent on Microsoft Windows
------------------------------
NewRelicDotNetAgent_{VERSION}_x64.msi
    MSI for installing the 64 bit version of the .NET agent (both Framework and Core/.NET 5+) on Windows.

NewRelicDotNetAgent_{VERSION}_x86.msi
    MSI for installing the 32 bit version of the .NET agent (both Framework and Core/.NET 5+) on Windows.

NewRelicDotNetAgent_{VERSION}_x64.zip
    Zip archive containing the 64 bit version of the .NET agent (both Framework and Core/.NET 5+) for Windows.

NewRelicDotNetAgent_{VERSION}_x86.zip
    Zip archive containing the 32 bit version of the .NET agent (both Framework and Core/.NET 5+) for Windows.


Assets for installing the agent on Linux
------------------------------------------
newrelic-dotnet-agent-{VERSION}.x86_64.rpm
    RPM package for installing the .NET Core agent on supported Linux platforms (RPM-based, x64/AMD64 CPU architecture).

newrelic-dotnet-agent-{VERSION}_amd64.deb
    DEB package for installing the .NET Core agent on supported Linux platforms (Debian-based, x64/AMD64 CPU architecture).

newrelic-dotnet-agent-{VERSION}_arm64.deb
    DEB package for installing the .NET Core agent on supported Linux platforms (Debian-based, ARM64 CPU architecture).

newrelic-dotnet-agent-{VERSION}_amd64.tar.gz
    Tar archive containing the .NET Core agent for supported Linux platforms (x64/AMD64 CPU architecture).

newrelic-dotnet-agent-{VERSION}_arm64.tar.gz
    Tar archive containing the .NET Core agent for supported Linux platforms (ARM64 CPU architecture).
